## Portafolio — German Gallo
Este repositorio contiene mi portafolio como desarrollador web, con proyectos que integran arquitectura modular, herramientas avanzadas y tecnologías modernas como Electron, Node.js y Monaco Editor.

## 🛠️ Tecnologías principales
- Frontend: HTML, CSS, JavaScript
- Backend / Runtime: Node.js
- Aplicaciones de escritorio: Electron
- Editor embebido: Monaco Editor
- Metodologías: BEM, POO
- Control de versiones: Git y GitHub

## 📂 Proyectos incluidos
# 🔹 estructura-proyectos-app
Generador de estructuras de carpetas y archivos basado en indentación.
Construido con Electron, Node.js y Monaco Editor. Permite crear proyectos completos de forma automática a partir de un esquema escrito por el usuario.
# 🔹 Project_web_cultures
Aplicación web modular para gestionar tarjetas e información de usuario.
Desarrollada con HTML, CSS, JS, arquitectura orientada a objetos y estilos organizados con BEM.

## 📌 Objetivo del portafolio
Mostrar mis habilidades técnicas mediante proyectos reales que integran desarrollo web, arquitectura modular, aplicaciones de escritorio y buenas prácticas de organización.
