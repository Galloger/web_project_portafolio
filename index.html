Proyecto Web Culturas

Aplicación web donde los usuarios pueden crear, editar, eliminar y dar “like” a tarjetas informativas sobre culturas antiguas. Cuenta con una interfaz dinámica y un sistema modular basado en Programación Orientada a Objetos

Funciones principales: • Crear, editar y eliminar tarjetas con imagen, título y descripción • Dar “like” con contador en tiempo real • Ver imágenes ampliadas en un modal • Crear y editar perfil de usuario (nombre, descripción y avatar)

Arquitectura: • Proyecto estructurado con clases independientes (Card, Section, Popup, UserInfo) • Código modular y reutilizable • Punto de entrada: index.js

Estilos: • CSS organizado con metodología BEM para mantener escalabilidad y orden

Tecnologías: HTML5 | CSS3 (BEM) | JavaScript (ES6+) | POO

Generador de Estructuras de Proyecto

Esta herramienta permite crear automáticamente estructuras completas de carpetas y archivos a partir de un esquema escrito con indentación. Está diseñada para acelerar la creación de nuevos proyectos sin tener que generar carpetas manualmente.

Funciones principales: • Generación automática de carpetas y archivos según la estructura escrita por el usuario • Editor integrado con Monaco Editor (estilo VS Code) • Instructivo accesible desde la aplicación • Compatible con Windows, Linux y macOS • Empaquetado con Electron Forge.

Cómo funciona?: 1. Escribes la estructura usando indentación (2 espacios por nivel) 2. Presionas Generar estructura 3. Seleccionas la carpeta destino 4. La herramienta crea todo automáticamente en tu sistema de archivos.

Demostración: En el video puedes ver cómo la aplicación genera carpetas y archivos en tiempo real a partir del esquema escrito.

Tecnologías: Electron · Node.js · Monaco Editor · HTML · CSS · JavaScript
